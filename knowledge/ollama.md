# Ollama Model Selector

## Launching the GUI

**ollama_model_selector.py** is a Tkinter GUI to load Ollama models. Use when the user wants to switch or load an Ollama model.

**How to run:**
- Use `run_command`: `run_command("python ollama_model_selector.py")`
- Runs from project root. Requires: Ollama running on localhost:11434, `requests` installed.

**When to use:** User asks to "load an Ollama model" or "open Ollama selector." Launches the GUI; user picks the model in the window.
